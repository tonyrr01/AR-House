# Bolongo Concierge Mobile

App móvil en React Native con Expo para huéspedes, propietarios, proveedores y seguridad.

## Primeros flujos
- Inicio de sesión.
- Estancia activa.
- Pre-check-in.
- Solicitar servicio.
- Chat por solicitud.
- Estado de solicitudes.
- Evidencia de proveedor.
- Validación de pase QR para seguridad.
