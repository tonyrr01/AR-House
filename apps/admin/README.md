# Bolongo Concierge Admin

Panel web administrativo en Next.js para roles `admin` y `concierge`.

## Primeras pantallas
- Dashboard de solicitudes.
- Detalle de solicitud con chat, adjuntos y auditoría.
- Catálogo de servicios.
- Proveedores.
- Unidades y estancias.
- Pases de acceso y emergencias.

La app Next.js actual vive por ahora en la raíz del repo. Este directorio reserva la ubicación final del panel administrativo dentro del monorepo.
